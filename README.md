# C++ Style Developer Portfolio

This portfolio is designed in **C++ style**, making it highly readable and appealing to developers familiar with programming concepts. It showcases skills, experience, and projects using C++-like structures such as `class`, `struct`, `enum`, and functions.

## Key Features

- **Code-Like Presentation:**  
  - Personal info, skills, and experience are represented as `structs` and `enums`.
  - Functions are used to describe work experience and education.
  - Projects are displayed using enums with clickable links.

- **Interactive Elements:**  
  - Email, phone, LinkedIn, GitHub, and location are clickable links.
  - Projects link directly to live repositories.

- **Organized Layout:**  
  - Two-column structure for easy navigation.
  - Left column: Profile & Skills.  
  - Right column: Summary, Info, Experience, Education, Projects, Certifications, Hobbies.

- **Developer-Friendly Design:**  
  - Familiar C++ structures make it intuitive for developers to scan your portfolio.
  - Clear categorization of technical and personal skills.

## Benefits for Developers

- Quickly understand your **skillset and experience**.
- Clickable links allow easy access to your **work and contact info**.
- Code-style design reflects **professionalism and attention to detail**.
- Ideal for tech recruiters and fellow developers looking for a **quick yet comprehensive overview**.
